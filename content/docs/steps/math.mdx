# simple math usage


```ts
import { JsonData } from "multi.dbx";



const db = new JsonData(); 

db.connect("database/users.json");



db.math("points", "+", 900)
```

<br />

# using nested


```ts
import { JsonData } from "multi.dbx";



const db = new JsonData(); 

db.connect("database/users.json");



db.math("main.users[0].points", "+", 900)
```
